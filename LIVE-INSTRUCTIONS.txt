DETAILS ON SECTION MANAGEMENT
=============================

Skills section:
- Implemented in components/SkillsSection.tsx.
- Each category is a separate horizontal scroll row.
- Categories: Graphic Design, UI/UX Design, Video Editing, AI & Automation, Web Basics.
- Each skill is a button inside the row.
- Clicking a skill opens a modal with:
  * Skill name
  * What it can do
  * Real project examples
  * Tools used
  * Experience level
  * Mini portfolio preview
- This matches the requested category-based horizontal scroll system and modal interaction.

Experience section:
- Implemented in components/ExperienceSection.tsx.
- Uses a vertical timeline layout with cards.
- Each item includes:
  * Role
  * Company or context
  * Duration
  * Short 1-2 line work summary
  * Skills used displayed as tag chips
- This creates a recruiter-friendly timeline with clear role context and outcomes.

Portfolio section:
- Implemented in components/PortfolioSection.tsx.
- Filter system with categories: All, Graphic Design, UI/UX, Branding, Logo, Social Media, Video, Motion, Web, AI.
- Each item shows a clean thumbnail area, title, category, and hover preview style.
- Clicking a card opens a full-screen modal with:
  * Large preview area
  * Project description
  * Problem → Solution structure
  * Tools used
  * Role
  * Year
  * Image gallery preview
  * Live link / contact button

Contact section:
- Implemented in components/ContactSection.tsx and components/ContactForm.tsx.
- Clean split layout with left contact info and right form.
- Form fields are only: Name, Email, Message.
- Designed for simple client outreach without extra fields.

Hero section:
- Implemented in components/HeroSection.tsx.
- Includes the required name and typing-role animation for:
  * Graphic Designer
  * UI/UX Designer
  * AI Enthusiast
- Buttons: Hire Me, View Portfolio, Download CV.
